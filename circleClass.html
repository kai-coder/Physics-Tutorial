CLASS

Classes are essential in keeping the code free of clutter and easy to modify. Without them we will have to resort to creating many variables which will quickly render the code unreadable.

To begin, we will have the circle class store the circle's position and radius like so:

class circleCollider:
    def __init__(self, x, y, radius):
        # Store circle attributes
        self.x = x
        self.y = y
        self.radius = radius

Keep in mind that we will add more variables and functions to this class in the future when necessary.

The first function that we will add to the circle class is the draw function that we will use to visualize the circle. The class should now look like this:

class circleCollider:
    def __init__(self, x, y, radius):
        # Store circle attributes
        self.x = x
        self.y = y
        self.radius = radius

    def draw(self):
        # Draw the circle
        circle(self.x, self.y, self.radius * 2)

We will now create two circles and set their position and radius. 

# Create two circles
c1 = circleCollider(0, 0, 20)
c2 = circleCollider(300, 200, 100)

Now we will set the first circle's position to the mouse position and draw the circles onto the screen, giving us our final code:
